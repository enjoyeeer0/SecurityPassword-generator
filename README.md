## ✨ О проекте

Чистый и честный генератор паролей, который работает **полностью на стороне клиента** — без серверов, без трекинга, без рекламы и без cookies.

- Генерирует криптографически стойкие пароли с помощью `window.crypto.getRandomValues()`
- Длина от 8 до 32 символов
- Выбор состава: заглавные, строчные, цифры, спецсимволы
- Копирование в буфер обмена
- Показывает примерную стойкость пароля
- жоооский тёмный дизайн
- Надеюсь нормально адаптировал под мобильные устройства (но это не точно)
- Развёрнут на **Vercel**
